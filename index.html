Experiencia Profesional.

BOSSTRAP

Bootstrap es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web.

FOUNDATION

Foundation es una familia de marcos front-end receptivos que facilitan el diseño de hermosos sitios web, aplicaciones y correos electrónicos receptivos que se ven increíbles en cualquier dispositivo.

BULMA

Bulma es un marco CSS ligero y de código abierto basado en Flexbox.

SEMANTIC

Semantic es un marco CSS con el objetivo de crear interfaces de usuario intuitivas.

Uikit

UIkit es un marco moderno y liviano con un gran énfasis en la modularidad para crear interfaces web.
